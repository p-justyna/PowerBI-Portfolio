# PowerBI-Portfolio
Interactive Power BI Dashboard showcasing global nuclear energy data and reactor insights

Opis projektu:
Interaktywny dashboard Power BI pokazujący dane o energii jądrowej i szczegóły reaktorów.
Celem projektu jest prezentacja umiejętności w przekształcaniu danych, tworzeniu miar DAX, interaktywnych wizualizacjach i czytelnym układzie dashboardu.

Funkcje dashboardu:
- Slicery filtrujące według kraju, typu reaktora i statusu
- Przycisk „Reset filtrów” przywracający domyślny widok
- Strony: podsumowanie, produkcja energii, szczegóły reaktorów, opis technologii
- Karty KPI, wykresy słupkowe, Gauge, tabele z detalami

Dane:
Wszystkie dane są przykładowe i anonimowe. Plik .pbix gotowy do otwarcia w Power BI Desktop.

----------------------------------------------

Project description:
Interactive Power BI dashboard showcasing nuclear energy data and reactor details.
The project demonstrates skills in data transformation, DAX measures, interactive visualizations, and clear dashboard layout.

Dashboard features:

- Slicers filtering by country, reactor type, and status
- “Reset filters” button to restore default view
- Pages: Overview, Energy Production, Reactor Details, Technology Description
- KPI cards, bar charts, Gauge, tables with detailed information

Data:
All data are sample and anonymized. The .pbix file is ready to open in Power BI Desktop.
